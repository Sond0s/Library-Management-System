# Library-Management-System

## A Windows Forms application built using:

- C#
- ADO.NET
- SQL Server
- 3-Tier Architecture
- Guna UI2 (First try)

## Features:

### Authentication
- User Login

### Books Management
- View Books
- Add Book
- Update Book
- Delete Book
- Search Books

### Members Management
- Add Member
- Update Member
- Delete Member

### Borrowing System
- Borrow Books
- Return Books
- Fine Calculation

### Architecture

Presentation Layer (WinForms)
↓
Business Logic Layer (BLL)
↓
Data Access Layer (DAL)
↓
SQL Server Database

## Database

### Tables:
- Users
- Books
- Categories
- Members
- BookCopies
- Borrowings
- Fines
